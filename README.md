# 📰 News Portal — Version 1

A static news portal website built with vanilla HTML, CSS, and JavaScript. This was one of my earliest web projects, originally created as a school news portal named **"Voices of Government Higher Secondary School"** and submitted as an academic project.

## 🔍 About

This project explores front-end fundamentals — structuring a multi-section news website, styling it with CSS, and adding interactivity through JavaScript. It was built entirely without any frameworks or libraries.
